Visa Canada

Posted on 2017-10-18

Since this is used to apply for a Canada Visa in China, this article will be written in Chinese.
出国开学术会议申请签证，因此申请的类别是business。

加拿大签证申请中心
地址：中国北京市东城区东直门南大街11号中汇广场A座12层

提交文件列表

  • 护照以及护照的复印件
  • 在读证明
  • 加拿大会议方的邀请信
  • 身份证正反两面复印件
  • 户口本复印件
  • 最近六个月内拍摄的护照尺寸相片（白底彩照，尺寸为3.5X4.5CM,2寸），照片背面需要写上自己的姓名和出生年月日
  • 材料清单表 IMM 5484
  • 临时居民访问签证申请表 IMM 5257
  • 家庭信息表 IMM 5645
  • 教育就业细节表
  • 学校的派遣证明原件（中英文），不同学校的办理手续不一样 学校出具了资助证明
  • 资产证明，去银行冻结了一部分资金 (3w)

9月11日提交材料，10月10日官网查询状态更新，11日上午拿到护照。不同地区申请所需市场不一样，仅供参考。

backup your Hexo files on Github

Posted on 2017-10-18 | In Tech

Since you have create your website using github. Here is the way that you backup your files so that you can write your blog anywhere.

Steps

  1. clone your github repository

  2. create a new branch named hexo and copy files here

  3. push to Github

  4. generate and deploy your website

Hence, you have a repository. Branch master is used to store the website and branch hexo is used to store the codes that generate the website.

update your website on a new computer

  1. clone your github repository and switch to branch hexo

  2. install necessary things
    install Node.js

  3. Edit your files

  4. Update files on Github

  5. Deploy your website

Merge two picutres

This code will help you to merge two pictures.
One image is in PNG form, and the other is in JPEG form. The PNG image has 4 channels, rgba, and the JPEG image only has 3 channels, rgb. By merging them together and you can get the final result, which seems to be taken by you.

requirements

  • python

workflow

  1. read the PNG image Im and get rgba data;
  2. read the JPEG image as I and get rgb data;
  3. resize I to I', which is the smaller one as shown above;
  4. blur the image I using gaussian kernel;
  5. merge those images together according to the alpha channel data of Im
